Overview

Notepad++ is a free source code editor and Notepad replacement that supports several languages. Based on the editing component Scintilla, it is written in C++ and uses pure Win32 API and STL to provide higher execution speed and smaller program size.

Its features include:

  • Syntax Highlighting and Folding
  • User Defined Syntax Highlighting and Folding
  • Customizable GUI
  • Word completion, Function completion and Function parameters hint
  • Tab interface
  • Document Map
