Overview

Internet Explorer(since v10) (formerly Microsoft Internet Explorer(v6 & earlier) and Windows Internet Explorer(v7, 8, 9), commonly abbreviated IE or MSIE) is a series of graphical web browsers developed by Microsoft and included in the Microsoft Windows line of operating systems. Internet Explorer was once the most widely used web browser, attaining a peak of about 95% usage share by 2003. It is based on Trident layout engine and Chankra JavaScript engine. The browser is discontinued, but still maintained.

On March 17, 2015, Microsoft announced that Microsoft Edge would replace Internet Explorer as the default browser on its Windows 10 devices.
