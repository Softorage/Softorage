Overview

eM Client is an e-mail client for sending and receiving e-mails, managing calendars, contacts, and tasks. It was developed as a user-friendly alternative to existing e-mail clients and calendar solutions. It supports PGP encryption, backup, table editor, image editing and auto-replies for Gmail. eM Client requires Microsoft .NET Framework 2.0 installation.
