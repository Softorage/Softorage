Overview

Thunderbird is an email, news feed, RSS feed, and chat (XMPP, IRC, Twitter) client. Its features include message management; junk filtering; extensions and themes; standards support: POP,IMAP, LDAP, S/MIME, OpenPGP; file formats supported: mbox, maildir; file linking; HTML code insertion; cross-platform support for Linux, macOS, Windows, with unofficial ports for FreeBSD, OpenBSD; localization; and security.
